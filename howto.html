How to Take the Quiz

Step 1: Choose Number of Questions

Select how many questions you'd like to answer:

  • 10 Questions - Quick quiz (approximately 5 minutes)
  • 20 Questions - Comprehensive quiz (approximately 10 minutes)

Step 2: Select a Category

Choose one of the four available categories:

  • JavaScript Fundamentals - Test your knowledge of variables, functions, loops, and core JavaScript concepts
  • UI/UX Design - Challenge yourself on design principles, accessibility, and user experience
  • HTML & CSS Basics - Review HTML tags, CSS selectors, properties, and structure
  • Git & GitHub - Quiz yourself on version control concepts, commands, and workflows

Step 3: Answer the Questions

  • Read each question carefully
  • Click on your chosen answer from the multiple-choice options
  • Your selection will be highlighted
  • Use the Next button to move to the next question
  • You can use the Previous button to review earlier questions if needed

Step 4: Submit Your Quiz

  • Once you've answered all questions, click the Submit button
  • Don't worry - you'll be asked to confirm before submitting

Step 5: View Your Results

After submission, you'll see your results page showing:

  • Total Score - Your score as a percentage
  • Correct Answers - Number of questions you answered correctly
  • Incorrect Answers - Number of questions you got wrong
  • Answer Review - See all questions with:
    • Your answer
    • The correct answer
    • Detailed explanation to help you learn

Step 6: Try Again (Optional)

  • Click Retake Quiz to start a new quiz with different questions
  • Click Home to return to the main page
